  • La herramienta está desarrollada en Java, versión 10 o superior, por lo que necesitarás tener Java instalado en tu sistema.

  • Los drivers de la tarjeta gráfica se recomiendan, estén actualizados. Tanto la extracción de la cadena de capacidades del monitor como el envio de señales se realizan desde la gpu,
    una tarjeta gráfica antigua o desactualizada puede no ser compatible.


  • Si al intentar cambiar la entrada salta esta ventana de la imagen superior, indica que tu monitor no cuenta con DDC, por lo que no se puede controlar mediantes codigos VCP, unicamente podras apagarlo desde el programa.

  • Es importante que Java tenga permisos de administrador, ya que se utiliza la API User32 de Windows para comunicarse con el monitor. Los comandos VCP (Virtual Control Panel) hacen uso de JNA (Java Native Access) para interactuar directamente con User32.

  • Si encuentras algún problema o tienes alguna pregunta sobre la compatibilidad de tu monitor, no dudes en ponerte en contacto.
