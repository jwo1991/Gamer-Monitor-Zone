  • The tool is developed in Java, version 10 or higher, so you will need to have Java installed on your system.

  • It is recommended that your graphics card drivers are up to date. Both the extraction of the monitor's capabilities string and the sending of signals are performed from the GPU. An outdated or old graphics card may not be compatible.

  • If an error message appears while extracting the capabilities string, or if it shows that the monitor is not supported, then the monitor will not be compatible, the same goes for the brightness level label, if it says not supported.

  • It is important that Java has administrator privileges, as the Windows User32 API is used to communicate with the monitor. The VCP (Virtual Control Panel) commands use JNA (Java Native Access) to directly interact with User32.

  • If you encounter any problems or have any questions about the compatibility of your monitor, please do not hesitate to contact us.
