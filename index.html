The adaptive mode is designed to detect full-screen windows and analyze GPU usage. This system differentiates between games, which typically use more than 50% of GPU capacity during their initial load, and movie playback, by recognizing specific video player names. Although not an extremely complex system, it is efficient enough to manage graphic resources effectively. Having the latest GPU drivers from the manufacturer is generally sufficient. If the application doesn't work as expected, please follow these steps and checks.

  • Driver Installation:
    • - Ensure that the appropriate NVIDIA or AMD drivers are installed.
    • - For NVIDIA, install the driver that includes nvidia-smi.
    • - The application integrates AMD's ADLX library for GPU monitoring, so no additional action is needed by the user. Compatibility begins with AMD Radeon graphics cards.
  • Adding Paths to System PATH:
    • - Open the Control Panel and select "System and Security".
    • - Click on "System" and then "Advanced system settings".
    • - In the "Advanced" tab, click on "Environment Variables".
    • - In the "System variables" section, select the "Path" variable and click "Edit".
    • - Add the program path at the end of the list, ensuring each path is separated by a semicolon (;).
    • - For example, if nvidia-smi is located in C:\Program Files\NVIDIA Corporation\NVSMI, add this path to the PATH variable.
  • Verification of Installation:
    • - Open a command prompt (CMD) and type nvidia-smi for NVIDIA or verify that AMD ADLX monitoring is active.
    • - If the nvidia-smi command executes successfully, you will see the GPU information in the console.
    • - If you receive an error, verify that the paths are correctly configured in the PATH variable.
