MAIN WINDOW



  1. Turn off the monitors.
  2. Switch between different inputs.


SETTINGS AND VCP WINDOW



    Settings:

  1. Start the program by selecting the desired monitor (where the program will start).

  2. Customize input names and add icons to distinguish them.

  3. Change the main strip image.

    4. VCP:

  4. Extract VCP Codes button: the first time you use the program, click the button to extract the monitor codes and set them up. If an error window pops up, your monitor is not compatible with DDC or VCP control codes. (It may fail sometimes due to communication issues with the monitor. Try multiple times in case of failure, but click only once. The monitor might take time between queries, and multiple clicks can cause extraction errors.)

  5. View string capabilities button: you can see the entire capabilities string of your monitor, which is in hexadecimal format.


  6. The first time the application is launched, a configuration file is created in the executable root, which should not be deleted.

    7. BRIGHTNESS:

  7. Adjust the brightness by sliding and apply, the monitor may not be compatible, it will communicate with a pop-up window.





FINAL USE



  1. The monitor where the input change will occur will be the primary one in Windows.

  2. You can check this in Display Settings; the one labeled with the number 1.

  3. Turning off works for all monitors at once.



For the program to work correctly, it is ideal to have more than one monitor, as it is designed to work with multi-monitor setups. If you have only one, you will need to manually change the input using physical buttons after switching inputs, which this tool aims to avoid. Imagine performing tasks on your computer, such as data conversion, downloads, or video rendering, and using your main monitor. In that case, you could have a console or an Android TV device connected to the same monitor and switch the input directly from the second one. This way, you can keep controlling what you do on the computer on the secondary monitor and switch back to the main input whenever you want, avoiding rear or side physical buttons, especially in hard-to-reach positions when both monitors are together or when they are wall-mounted or on desk arms.

You can also have two towers on the same monitor and use the tool on each PC to switch the input or turn it off.
