Compatibility About Support Licenses
Java Requirements

The tool is developed using Java, version 10 or higher. Ensure that Java is installed on your system. Administrator privileges are required for proper operation.

.NET Requirements

Ensure that .NET is also updated to the latest version for proper monitoring functionality.
Operating System

Supported only on Windows (Windows 10 or higher) and requires a 64-bit version for optimal performance. The tool is not compatible with macOS or Linux.
Internet Connection

Required to download game metadata and update news from RSS channels.
Driver Updates

Ensure your graphics drivers are up to date for full compatibility. Both the extraction of monitor capabilities and signal transmission rely on the GPU, and outdated graphics cards may not function properly.
Monitor Compatibility

Supports up to four monitors with a maximum of three input parameters. If more are present, only the first three will be utilized. If you can read brightness, contrast, or saturation levels but cannot apply changes, this may be due to your monitor's factory settings allowing read-only permissions without enabling write access.
Tool Integrity

The tool is unsigned, as it is a personal project, and obtaining a digital signature incurs significant costs.
Known Limitations

May not function correctly if other system monitoring tools are running simultaneously.

It is recommended to use the free tool to check the compatibility of all monitors with the software. Based on the compatibility results, users can assess whether the incompatible features will pose a problem for their intended use of the tool.
