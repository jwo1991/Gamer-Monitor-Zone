Provides detailed real-time data on CPU usage, including core performance, clock speeds, and temperature levels. Tracks individual core utilization, allowing for monitoring of total CPU consumption and identification of resource-intensive processes.

Compatibility

  • The tool is developed using Java, version 10 or higher, which requires you to have Java installed on your system.

  • Only supported on Windows (Windows 10 or higher) and on dedicated AMD and NVIDIA graphics, but not on Intel GPUs or integrated graphics.

  • The tool is unsigned, as it is a personal project and obtaining a digital signature incurs significant costs.

  • NVIDIA: Requires nvidia-smi to be installed. This tool usually comes with the official drivers. To check if it's installed, open a command prompt and type nvidia-smi. If it's recognized, it's ready to use.

  • AMD: The tool already includes the ADLX library, so no additional installation is required.

  • Administrator privileges are required for proper operation.

  • Portable version: The tool does not require installation and can be run as a portable application.

  • Requires a 64-bit version of Windows for optimal performance.

  • Ensure that your graphics drivers are up to date for full compatibility.

  • May not function correctly if other system monitoring tools are running simultaneously.

GENERAL MAINTENANCE

  • Routine Cleaning: Conduct a thorough cleaning of the system at least every three months. This includes removing accumulated dust from the case, fans, and filters. Use compressed air to clean internal components and a soft cloth for external surfaces. Ensure to clean the CPU and GPU coolers, as dust can obstruct airflow and lead to overheating.

  • Environmental Considerations: Keep the PC in a cool, dry environment, away from direct sunlight and sources of humidity. An ideal ambient temperature range is between 15°C and 25°C (59°F to 77°F). Additionally, ensure there is adequate space around the PC to allow for proper airflow.

  • System Performance Monitoring: Utilize SMM to track CPU and GPU performance. Regularly review reports generated by SMM to ensure CPU temperatures remain within the ideal range of 30-70°C under load and that GPU temperatures stay between 60-85°C. Pay attention to any alerts regarding elevated temperatures or excessive resource usage.

  • Software Maintenance: Keep your operating system and applications up to date to ensure optimal functionality and security. Regularly check for BIOS and firmware updates that may enhance stability and performance. Review SMM drivers and software to ensure you are using the latest version.

  • Surge Protection: Protecting the system from power surges is critical. Use a high-quality surge protector and consider investing in an uninterruptible power supply (UPS) to safeguard against power outages and fluctuations that could damage your PC components.

CPU MAINTENANCE

  • Temperature Monitoring: Use SMM to monitor CPU temperatures. Ensure temperatures remain below 80°C under load. Be vigilant for any sudden temperature spikes and take immediate action if thresholds are exceeded.

  • Avoid Overclocking: If you decide to overclock, ensure you have a clear understanding of the CPU specifications and temperature limits. Use SMM to monitor performance and temperature changes during overclocking.

  • Thermal Paste Replacement: It is advisable to replace thermal paste every 2-3 years or each time the CPU cooler is removed. Utilize high-quality thermal paste to ensure efficient heat transfer, which is crucial for maintaining optimal CPU temperatures.

  • Check Adequate Cooling: Periodically inspect the CPU cooler to ensure it is mounted correctly and functioning properly. Ensure that the fans are spinning freely and that airflow within the chassis is sufficient.

  • Power Supply Check: Ensure the power supply unit (PSU) meets the requirements of your CPU. Verify that the PSU has an 80 Plus certification to ensure efficiency and stability, which is important to avoid voltage fluctuations that could affect performance.
×

GPU MAINTENANCE

  • Temperature Monitoring: Use SMM to monitor GPU temperatures. Keep temperatures below 85°C under load. If you notice a significant increase, consider inspecting the cooling system and cleaning the GPU fans.

  • Avoid Overclocking: Overclocking the GPU can be risky; ensure you understand the manufacturer’s specifications. Use SMM to monitor performance stability while applying any overclocking settings.

  • Thermal Paste Cleaning: Replace thermal paste on the GPU every 2-3 years, especially if the GPU is used for high-performance tasks. This helps maintain good thermal conductivity and prevents overheating.

  • Inspect for Graphical Artifacts: Monitor for any graphical artifacts or performance issues using SMM. The presence of these can indicate overheating or hardware failures, requiring immediate attention.

  • Driver Updates: Keep GPU drivers updated. SMM can assist you in checking the current version and alerting you about necessary updates to ensure compatibility and optimal GPU performance.

  • Power Supply Evaluation: Verify that the power supply meets the power requirements of the GPU, particularly for high-performance models. An 80 Plus Gold certified power supply is recommended for gaming and graphics-intensive systems.
