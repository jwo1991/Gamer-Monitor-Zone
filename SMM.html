Compatibility

  • The tool is developed using Java, version 10 or higher, which requires you to have Java installed on your system.

  • Only supported on Windows (Windows 10 or higher) and on dedicated AMD and NVIDIA graphics, but not on Intel GPUs or integrated graphics.

  • The tool is unsigned, as it is a personal project and obtaining a digital signature incurs significant costs.

  • NVIDIA: Requires nvidia-smi to be installed. This tool usually comes with the official drivers. To check if it's installed, open a command prompt and type nvidia-smi. If it's recognized, it's ready to use.

  • AMD: The tool already includes the ADLX library, so no additional installation is required.

  • Administrator privileges are required for proper operation.

  • Portable version: The tool does not require installation and can be run as a portable application.

  • Requires a 64-bit version of Windows for optimal performance.

  • Ensure that your graphics drivers are up to date for full compatibility.

  • May not function correctly if other system monitoring tools are running simultaneously.
